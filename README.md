# QuakeMap

此程序基于UC San Diego提供的程序框架。

程序使用了UnfoldingMaps库。
http://unfoldingmaps.org
---



## 💻程序设计
- 面向对象编程，使用继承创建出不同的Marker，并显示在UnfoldingMaps库提供的地图上。
- 标明每个地震可影响的城市范围。
